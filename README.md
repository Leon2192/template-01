# template-demo

Template base para invitaciones digitales de eventos hecho con React + Vite.

## Correr el proyecto

```bash
npm install
npm run dev
```

## Personalizacion rapida

La configuracion central esta en:

- `src/data/invitationData.js`

Desde ahi podes cambiar:

- fecha del evento
- portada
- textos
- direccion
- links
- datos bancarios fake
- QR y acciones
